📊 BOE Model — Two Modes
Mode A — Know the Price

Input asking price + deal characteristics → Claude fills assumptions → outputs IRR, equity multiple, cash-on-cash, NOI, cost per unit

Mode B — Solve for Price

Input target return metrics (e.g. 15% IRR, 1.8x equity) → Claude back-solves → outputs max purchase price you can pay to hit those returns
