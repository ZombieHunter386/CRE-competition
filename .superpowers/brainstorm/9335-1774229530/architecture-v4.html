👥 Deal Contacts — CRM Layer
Contact Roles (multiple per deal)
🏠 Owner / Seller 🤝 Listing Broker 🏦 Buyer's Broker ⚖️ Attorney 🏗️ Contractor + Custom Role
Per Contact Fields
  • Name, role, company
  • Phone, email
  • Mailing address (owner auto-filled from tax data)
  • Notes / last contacted
  • Linked to broker email draft
🔗 Tax data auto-fill: When PIN is entered, county assessor API pulls owner name + mailing address automatically. User can confirm or override before saving.
📊 BOE Model — Two Modes
Mode A — Know the Price

Input asking price → Claude fills assumptions → outputs IRR, equity multiple, cash-on-cash, NOI, cost per unit

Mode B — Solve for Price

Input target returns (e.g. 15% IRR, 1.8x equity) → Claude back-solves → outputs max purchase price
